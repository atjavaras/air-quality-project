# Air Quality Analysis Project

Analysis of CO, PM2.5, and O3 pollutant concentrations in Durham, NC from 2018-2020.

## Files
- `project2.Rmd` - Main analysis document
- `project2_script.R` - R script with analysis code
- `data-6.zip` - Raw data files
- `data/` - Extracted CSV files

## How to Run
Open `project2.Rmd` in RStudio and knit to Word.
